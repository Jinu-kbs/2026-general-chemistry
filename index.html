과목 정보

과목명 일반화학 I (General Chemistry I)
대상 화학공학과 / 에너지공학과 1학년
교재 Chemistry: The Central Science (Brown) 15판, ebook ok
범위 1장 ~ 12장

수업 방식

  • 개념 중심 + 예제 중심: 핵심 개념을 대표 예제를 통해 학습합니다.
  • 토론식 / 발표식 수업: 조별 토론, 개별 발표, 질의응답을 통한 참여형 수업을 진행합니다.
  • TA(조교) 지원: 연구실 조교가 보충 설명, 과제 관리, 질의응답을 지원합니다.
고등학교 화학II를 이수하지 않은 학생도 따라올 수 있도록 기초 개념부터 연결하여 설명합니다. 그러나, 기준은 고교 과학과정을 모두 이수한 것을 기준으로 수업을 진행합니다.

출석

  • 전자출결을 사용합니다.
  • 교수자가 출결을 켜서 끄는 시점이 출석 인정 기준입니다. (1~2분 소요)
  • 전자출결 확인 여부는 매시간 본인 책임입니다. 전산 오류 등은 바로바로 확인하시기 바랍니다.
  • 지각은 전자출결 시점 이후부터 10분까지는 지각으로 처리합니다.
  • 변동이 필요하면 당일 교수자에게 메시지 등으로 문의 바랍니다.
  • 출석일 이후 일주일 이후에는 출석 변경이 원칙적으로 불가합니다.

과제 안내

  • 형태: 매주 주간일지를 작성하여 제출
  • 내용: 해당 주 학습 내용 정리 + 연습문제 풀이 + 개념 정리
  • 제출 형식: 웹 기반 산출물 (HTML 파일 또는 웹 페이지)
  • 제출 방법: 링크 제출 또는 파일 제출
마감: 매주 금요일 오후 6시 (마감 이후 제출은 인정되지 않습니다)
